# NMD Medical - Healthcare Appointment Scheduling

Hệ thống đặt lịch khám bệnh trực tuyến được phát triển bởi **Nguyễn Mạnh Dũng**. Dự án cho phép Bệnh nhân tìm kiếm bác sĩ và đặt lịch, đồng thời giúp Bác sĩ quản lý lịch trình làm việc.

## 🚀 Tính năng chính
- **Dành cho Bệnh nhân:** Đăng nhập, tìm kiếm bác sĩ theo chuyên khoa, chọn ngày/giờ và thanh toán qua Stripe.
- **Dành cho Bác sĩ:** Quản lý hồ sơ cá nhân, xem lịch khám trong ngày và lịch sử khám bệnh.
- **Bảo mật:** Sử dụng JWT cho xác thực và Bcrypt để mã hóa mật khẩu.

## 🛠 Công nghệ (Tech Stack)
- **Frontend:** ReactJS, CSS Modules.
- **Backend:** Node.js, Express.js.
- **Database:** MongoDB.
